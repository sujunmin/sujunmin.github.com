從 SQL Server 到 MariaDB - [0] 起始

2016-08-07
MSSQL/MariaDB

需求說明

公司因為 SQL Server 的支援快要到期，加上資料的等級與加值性不如預期，所以開始有從 SQL Server 轉到 MariaDB 的計劃。

測試環境

  1. SQL Server 2008 R2 64 bits (要汰換的)
  2. Oracle DB 12c (要汰換的)
  3. MariaDB 10.1.16 64 bits
  4. MariaDB 10.1.17 64 bits
  5. MariaDB 10.1.18 64 bits
  6. Windows Server 2012 64 bits (上面的資料庫都跑在這個上面)
  7. MariaDB ODBC 2.0.12 64bits
  8. Visual Studio 2015 Community

Windows Mobile手機與Google行事曆同步免費方案

  • Windows Mobile
  • http://schemas.google.com/blogger/2008/kind#post

我的手機(CHT9100)最常用的功能就是看行事曆，因為事情實在太多了。之前都是用迷版的Oggsync來做同步，不過自從Google行事曆的Protocols改變後，我的手機就只能同步行事曆內容，但是時間都不對(應該是對不到)。一直有一段時間無法同步行事曆。直到在網路上找到NuevaSync的免費服務，我的手機又活了過來。

現在Windows Mobile手機要與Google行事曆同步，我找到的大概有下列幾個方式：

  • 使用Google Sync服務。Google Sync使用Microsoft Exchange Service來做服務，手機端使用ActiveSync來更新，不過有他的限制：









    Google Sync for Windows Mobile uses the Microsoft Exchange ActiveSync protocol to synchronize the data on your phone with your Google Account. Google Sync currently supports synchronization with your Gmail, main Google Calendar, and the contacts in your My Contacts group.

    也就是只有Default那個行事曆會被同步，這對我來說(因為各專案或是其他事情有不同的行事曆)，不是太方便，所以不能用(但iphone與黑莓機用戶是可以同步全部的，真是氣人)。

 

  • 使用Oggsync的產品(beta到4.21，stable是4.19.4)來同步，不過Free版也是只能同步一個行事曆(Default那一個)，Pro版可以同步很多個。

  • ….
  • Sync Multiple Calendars (Pro version)
  • … 




至於Pro版要多少錢?OggSyncProYearly每年要$29.95USD…不太便宜，每年還要繳錢，有一些優惠啦，不過我不需要那麼多，不過我是覺得很蠻不錯的服務(我都拿它加MortScript就可以做排程與開關網路了)。



 

 

  • GooSync也是根據你的機子是哪一個，灌對應的軟體，之前有免費版的，不過現在只有Lite版跟Premium版，Lite版比較便宜，但是只能同步一個，Premium版可以同步很多個，但是終身的要£39.95，真是很貴。

 

  • Google行事曆->Microsoft Live行事曆->Windows Mobile Live Service同步，用ical對傳，不過現在Windows Mobile Live Service還沒支援行事曆同步(這篇，當然下面的Post直接酸微軟這種事情也不做)。

 

  • 使用Google行事曆的ical同步(因為我不會改手機端的資料，所以只要讀就可以了)，我找到GCalSync，GMobileSync，iCalParse，其中iCalParse是有常更新的程式，不過最新版的Google行事曆Protocol還是不支援，一堆錯誤訊息(還沒法控制要轉多少事件，害的我手機要轉下8xx個事件，都當機好幾次)。

 

  • 在網路上找ical同步的資料，忽然看到不知道哪篇說為何不使用NuevaSync來做同步，想說連過去看看，結果看到竟然支援11個行事曆同步，實在太高興了，因為我的還沒那麼多，趕快來試試，現在在我的手機配合MortScript就可以排程同步了，希望這個免費服務可以一直下去。

     

這篇的剩下部分，就是使用NuevaSync的心得，供大家參考(以下圖片皆各所屬公司所有)。

首先先點Get a free account，填完就按Submit(他的辨識圖說實在真難看懂)。


接著你的信箱會收到確認信，點選裡面的連結完成認證。

 

接著到這邊登入，打入你的帳號密碼，登入，進入以後你可以對你的行事曆做設定(我的是已經設好了)。可以看得出來，他的服務不只有同步行事曆一項，還有聯絡人、Email、Tasks等。

首先選擇Calendar的change，會請你選是要同步哪邊的行事曆，免費版只有Google可以選。


(以下部分流程為與Google要求同意連結的過程，以聯絡人同步來做展示，圖上的是聯絡人)接下來是選要連結的Google帳號(xxx@gmail.com)，輸入完成後，點擊Request Account Access。然後會看到Google的認證網頁，選一個你要連結的Google網站，再確認是否要連結，就完成認證連結了。


按下Continue以後，回到首頁，選擇Setup，設定你要同步的行事曆，按下Submit就完成需同步的行事曆設定。

接下來就是設定手機上的ActiveSync設定。首先打開ActiveSync，選擇功能表的新增伺服器來源，首先先打入Google帳號(xxx@gmail.com)，記得嘗試自動偵測Exchange Server設定不要打勾。打完選下一步。

接下來輸入Exchange Server位址：www.nvevasync.com，下一步繼續(SSL要記得打勾，這是預設的)。

接著輸入之前申請的帳號(不是Email的另一個，Username)，密碼，網域打notused，按下一步，選擇同步行事曆(若你有設定其他同步，也可選一下)，按完成，就完成設定。


 

接著ActiveSync會自動同步，看看結果就知道有沒有設對啦。

配合MortScript就可以排程同步+開關網路了，下面是我寫的Script(中華電信為連線的Profile，只有一個Exchange Server要連，所以直接按軟右鍵)：

If (NOT InternetConnected(“http://google.com/m“))
 connect(“中華電信(internet)”)
 Sleep 500
EndIf
Run( “\windows\async.lnk” )
WaitForActive( “ActiveSync”, -1 )
Sleep 3000
SendSpecial( “LeftSoft” )

If (InternetConnected(“http://google.com/m“))
 disconnect(“中華電信(internet)”)
 Sleep 500
EndIf

最後還是希望這個服務能夠一直下去，要不然我的手機又要到末日了。
